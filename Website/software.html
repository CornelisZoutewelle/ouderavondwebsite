Office 365

Office 365 is een pakket van verschillende programma’s en internetdiensten die gezamenlijk voor één prijs aangeboden worden door Microsoft. In dit pakket vinden we natuurlijk een aantal programma’s die voor veel mensen al bekend zijn en veel gebruikt worden. We hebben het dan over Microsoft Word, Microsoft Excel, Microsoft Outlook en Microsoft Powerpoint. Er zijn verschillende pakketten die dan ook verschillende programma’s aanbieden.

Visual Studio Code

Leren te coderen is intimiderend, dus stel jezelf in voor succes met een tool voor jou gebouwd. Visual Studio Code is een gratis coderingseditor die u helpt snel te beginnen met coderen. Gebruik het om te coderen in een programmeertaal, zonder van editors te wisselen. Visual Studio Code biedt ondersteuning voor vele talen, waaronder Python, Java, C ++, JavaScript en meer. Klaar om te beginnen?

EduArte

Eduarte is het marktleidende onderwijsinformatie systeem voor het MBO. Het biedt vlekkeloze informatievoorziening voor student, docent, administratie en management. Daarmee is Eduarte een belangrijke stap naar een totale integratie van informatiehuishouding. Eduarte brengt leren en toetsen, begeleiden en verantwoorden, beheren en faciliteren samen.

Canvas

Canvas is het communicatieplatform voor vakken van de VU. Hier kun je studenten bijvoorbeeld op de hoogte brengen van belangrijke zaken, je kunt bestanden met ze delen en hier kunnen ze opdrachten inleveren. Het is belangrijk dat studenten daadwerkelijk alles kunnen vinden wat ze nodig hebben op deze Canvascursussen. Daarom is het project Canvas Studentvriendelijk begonnen: hoe kun je Canvas op zo’n manier gebruiken dat studenten nooit iets missen en makkelijk de juiste informatie kunnen vinden? Als docent hoef je dan nooit meer te horen dat studenten iets niet wisten, en studenten hoeven geen tijd te verspillen door te zoeken; tijd die ze ook hadden kunnen gebruiken om te studeren.
